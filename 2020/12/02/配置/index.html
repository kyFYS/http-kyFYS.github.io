语法

Ctrl + 1-6 分别为 1-6级标题

Ctrl + b 加粗

Ctrl + I 斜体

Ctrl + u 下划线

Ctrl + alt + s 删除线

Ctrl + k 超链接

Ctrl + T 表格

注释

三个—创建分割线

Ctrl + shrift + q 引用

双击enter退出引用

shrift + CTRL + I 创建图片路径

-+空格 创建列表 tab 处理缩进
